CORE HOOKS / 核心 Hooks
useThreadStream
function useThreadStream({ threadId, context: { mode, thinking_enabled, subagent_enabled, ... }, isMock, onStart, onFinish, onToolEnd }): [thread, sendMessage, isUploading]
核心聊天 Hook：管理线程状态、发送消息、处理 SSE 流式响应、追踪上传状态
useThreads
function useThreads(params: { limit, offset, sortBy, sortOrder }): { threads, isLoading, error }
线程列表 Hook：分页加载、排序、错误处理
getAPIClient
function getAPIClient(isMock?: boolean): LangGraphClient
API 客户端单例：连接 LangGraph Server，提供 threads/runs/stream 方法
