DeerFlow 系统总览
Full-Stack Super Agent Harness
编排子代理、记忆和沙箱，通过可扩展技能执行复杂任务
What is DeerFlow?
DeerFlow 是一个全栈"超级代理框架"，它将子代理、记忆系统和沙箱环境整合在一起，通过 LangGraph 工作流引擎编排复杂的任务执行流程。
Core Features
Lead Agent: LangGraph 状态机
Subagents: 并行任务委托
Sandbox: 代码执行隔离
Memory: 用户记忆持久化
MCP: 工具协议集成
Skills: 可扩展技能系统
Guardrails: Pre-tool-call 授权
Circuit Breaker: 速率限制保护
Checkpointer: 状态持久化
Streamdown: 流式 Markdown 渲染
Design Philosophy
Harness/App 分离: 可发布框架 vs 应用代码
中间件链: 按顺序执行的处理层
TDD: 测试驱动开发强制
统一入口: Nginx 反向代理
Gateway Mode: 嵌入式运行时选项
Config Versioning: Schema 升级追踪
