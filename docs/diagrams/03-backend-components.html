DeerFlow Backend 组件结构
Backend Architecture / Harness vs App Boundary
Dependency Rule App imports deerflow, deerflow NEVER imports app
app.*
→
deerflow.*
(单向依赖，测试: tests/test_harness_boundary.py)
HARNESS packages/harness/deerflow/ 可发布的 Agent 框架
agents/
deerflow.agents.*
Agent 系统：Lead Agent、中间件、记忆、子代理
lead_agent/agent.py → make_lead_agent()
lead_agent/prompt.py → apply_prompt_template()
middlewares/*.py → 12个中间件组件
memory/updater.py → MemoryUpdater
subagents/ → 子代理注册与执行
thread_state.py → ThreadState 状态定义
sandbox/
deerflow.sandbox.*
代码执行沙箱
local/local_sandbox.py
tools.py → bash, read_file
sandbox_provider.py
mcp/
deerflow.mcp.*
MCP 工具集成
client.py → MCPClient
tools.py → load_mcp_tools
cache.py → 工具缓存
models/
deerflow.models.*
LLM 模型工厂
factory.py → create_chat_model()
patched_openai.py
patched_deepseek.py
vllm_provider.py
tools/
deerflow.tools.*
内置工具集
builtins/task_tool.py → task()
builtins/present_file_tool.py
builtins/ask_clarification_tool.py
skills/
deerflow.skills.*
Skills 加载系统
loader.py → load_skills()
parser.py → SKILL.md 解析
installer.py → .skill 安装
runtime/
deerflow.runtime.*
运行时系统
runs/worker.py → run_agent()
runs/manager.py → RunManager
stream_bridge/ → SSE 桥接
config/
deerflow.config.*
配置系统：AppConfig、Agent 配置、Memory/Sandbox 配置
app_config.py → AppConfig (config.yaml 解析)
agents_config.py → Agent 配置
memory_config.py → MemoryConfig
sandbox_config.py → SandboxConfig
models_config.py → 模型配置
community/
deerflow.community.*
社区贡献工具
ddg_search/ → DuckDuckGo 搜索
jina_ai/ → Jina Reader
tavily/ → Tavily 搜索
image_search/ → 图片搜索
aio_sandbox/ → Docker 沙箱
APP app/ 应用层，不可发布
gateway/
app.gateway.*
FastAPI Gateway API
app.py → FastAPI 应用入口
services.py → start_run(), sse_consumer()
deps.py → 依赖注入
routers/agents.py → /api/agents
routers/threads.py → /api/threads
routers/memory.py → /api/memory
routers/mcp.py → /api/mcp
routers/skills.py → /api/skills
routers/uploads.py → 文件上传
channels/
app.channels.*
IM 渠道集成
service.py → ChannelService
manager.py → ChannelManager
message_bus.py → 消息总线
feishu.py → 飞书集成
slack.py → Slack 集成
telegram.py → Telegram 集成
