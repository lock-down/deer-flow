SSE Event Types 流式响应事件格式
EVENTDESCRIPTION
metadata运行元数据 (run_id, thread_id)
values状态快照 (完整 ThreadState)
messages消息增量 (AIMessagechunk)
tool_calls工具调用事件
artifacts工件生成通知
error错误信息
end流结束信号
Stream Modes
stream_mode: ["values", "messages"] values → 每次状态变更发送完整状态 messages → 消息增量流 (更高效)
